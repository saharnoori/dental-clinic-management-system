## Description
This project is a Dental Clinic Management System developed as the final project for CS50.

The application helps dental clinics manage patient information, appointments, and dental services such as fillings, implants, and X-ray records.

The system is built using Python with a Tkinter graphical interface and uses SQLite3 for local data storage. The application works offline and is designed for small and medium dental clinics.

## Features
- Patient registration and management
- Appointment scheduling
- Dental services management (fillings, implants, X-rays)
- Simple and user-friendly interface

## Technologies Used
- Python
- Tkinter
- SQLite3
